# Modelos de isoladores
Aqui eu deixarei modelagem em models () para modelos de isoladores. Dispenso a curva tempo-tensão, trabalharei com alguns modelos LPM (Leader Progress Model) e o DE (Disruptive Effect ou também chamado de método integrativo). A ideia de trabalhar com models é utilizar uma linguagem de programação para modelagem de parte do sistema no ATP.
