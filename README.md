# Lightning-Tools

This repository provides tools for studying electromagnetic transients associated with lightning discharges. Its main objective is the modeling of equipment in ATP/ATPDraw, which is my primary area of interest. The main reference is CIGRE Technical Brochure 839, published in 2021, which presents updated procedures, models, and considerations for lightning performance studies [1].

## Insulator Models

The insulator models adopted in this project, including the *Leader Progression Model* (LPM) and the *Disruptive Effect* (DE) model, have been widely considered in the literature for assessing the lightning performance of transmission lines.

## References

[1] C. S. Engelbrecht *et al*., “Procedures for Estimating the Lightning Performance of Transmission Lines—New Aspects,” CIGRE Working Group C4.23, Paris, France, Technical Brochure 839, Jun. 2021.
