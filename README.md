# Lightning-Tools

Aqui compartilho algumas ferramentas para estudos de transitórios eletromagnéticos associados a descargas atmosféricas. O principal objetivo é a modelagem de equipamentos no ATP/ATPDraw, minha principal área de interesse.

## Modelos de isoladores

Os modelos de isoladores adotados neste projeto, incluindo o *Leader Progression Model* (LPM) e o *Disruptive Effect* (DE), têm sido considerados na literatura para a avaliação do desempenho de linhas de transmissão frente a descargas atmosféricas. Uma referência atual é a Brochura Técnica 839 do CIGRE, publicada em 2021, que apresenta procedimentos, modelos e aspectos atualizados para a estimativa desse desempenho [1].

## Referências

[1] C. S. Engelbrecht *et al*., “Procedures for Estimating the Lightning Performance of Transmission Lines—New Aspects,” CIGRE Working Group C4.23, Paris, France, Technical Brochure 839, Jun. 2021.
